Write Tymeac Statistics

This frame class requests On-Request Statistics of the Tymeac Server environment.

Buttons

Write Statistics -- Press to request On_request Statistics of the Tymeac Server environment.

Message Block at bottom : -- This is an information message:

Connection failure --   The RMI connection failed or the Tymeac Server returned an invalid message.

Tymeac shutting down --  Tymeac Server is in shut down mode.

DBMS Stats Table is not available --  There is no Stats Table for this session.  Either the table does not exist or the Configuration File does not specify a Statistics Table or no Configuration File is in use.

DBMS error  --  There was some type of DBMS error.  The console may contain additional information.

Tymeac stats were written --  The request was successful.
