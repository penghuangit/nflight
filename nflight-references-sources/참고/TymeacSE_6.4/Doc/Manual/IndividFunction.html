Tymeac Individual Function

The purpose of this Class is for use with the User Class, TymeacUserFunctions.  This Class contains the description of a single Tymeac Function.

Instance Fields

These fields are the same as found in the DBMS Function Table maintenance Class, TyFuncMaint.

String name - The name of the Function. See also how to name Functions, and the Function name checking class, FunctionChecker in the sample code.

String oa_name - The name of the Output Agent Queue, when needed.

String[] list - The list of all the Tymeac Queues for this Function.

int nbr_que - The total number of Queues in the above list.
