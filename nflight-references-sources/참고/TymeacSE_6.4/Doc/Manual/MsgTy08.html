Messages: General Tymeac Component Ty08

This section details the General Tymeac Component messages.

(P) Printed to System.out.println(), (this is optional).
(L) Sent to the Tymeac Log.
(N) Sent to the Tymeac Notification Function.

001 I (P) (L)

Tymeac Server Notification Disabled, returned async=xxxx

The server attempted to write to the Notification Function. Tymeac uses an asynchronous Request (asyncRequest()) internally. The request return that which came back in xxxx.

Notification is disabled.

You may resolve the issue and either reset Notification or set a new Notification Function with the Reset/New Notification GUI.

101 I (P) (L)

Tymeac Server Logging re-verified

During the current session the logging service failed for some reason. There were messages previously written describing this failure.

A user used the Reset/New logging function to reset logging. This was successful.

201 I (P) (L)

Tymeac Server Statistics re-verified

During the current session the Statistics service failed for some reason. There were messages previously written describing this failure.

A user used the Reset/New Statistics function to reset Statistics. This was successful.

301 I (P) (L)

Tymeac Server Notification Function re-verified

During the current session the Notification service failed for some reason. There were messages previously written describing this failure.

A user used the Reset/New Notification function to reset Notification. This was successful.
