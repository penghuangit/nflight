Tymeac™ Technical Reference

Tymeac™ (pronounced Tie-Mack) is an acronym for asYnchronous Task Manager and Easy Assembly of Components.

Contents

  • Getting Started  --  Product overview and instructions for setting up the environment.
  • Functionality  -- This is the section for developing your own systems.
  • Operation  --  This section is for using the Tymeac Product. Start up, shut down, etc.
  • Tymeac Classes  --  This section lists the all the classes that come with the Tymeac Product.
  • JavaDoc for selected classes -- Tymeac classes mentioned in this documentation.
     
  • Support  --  Product support
  • Index  --  Document index.

  Tymeac Classes

   Execution

  com.tymeac.base.TymeacInfo  
Singleton Class containing information on the currently executing Tymeac Server.

  com.tymeac.base.TymeacParm 
Parameter passed to the Tymeac Server.

  com.tymeac.common.TyVariables 
The Class containing the local RMI URL and Tymeac Name for binding.

   User Classes (non-DBMS)

  com.tymeac.serveruser.TymeacUserVariables  
The Class for the non-DBMS version of Tymeac for start up values.

  com.tymeac.serveruser.TymeacUserQueues  
The Class for the non-DBMS version of Tymeac for holding Queues.

  com.tymeac.base.TymeacIndividualQueue  
The Class is used by the above, TymeacUserQueues Class.

  com.tymeac.serveruser.TymeacUserFunctions 
The Class for the non-DBMS version of Tymeac for holding Functions.

  com.tymeac.base.TymeacIndividualFunction  
The Class is used by the above, TymeacUserFunctions Class.

   Frame Classes  The package prefix is: com.tymeac.client.jframe

  TyMenu  Display of the Frame Processes, in the list below.

  TyCfg  Configuration File Maintenance Class.

  TyAltSvr  Alter the running server options

  TyUserVariables  User Variables Class Maintenance Class.

  TyVariables  Tymeac Start Up Variables Class Maintenance Class.

  TyFuncData  Display of Functions loaded and times used during execution.

  TyFuncMaint  Function Table Maintenance Class.

  TyUserFuncMaint  User Functions Class Maintenance Class.

  TyNewCopy  Reload the Processing Application Class as a new version.

  TyNewRTNotify  Alter run time notification.

  TyNewRTLog  Alter run time logging.

  TyNewRTStats  Alter run time statistics.

  TyOverall  Display the overall load and Queue names with attendant threads

  TyQueData  Display/alter of the elements of a Queue during execution.

  TyQueMaint  Queue Table Maintenance Class.

  TyUserQueMaint  User Queue Class Maintenance Class.

  TyQueThd  Display/disable/enable of the threads of a Queue during execution.

  TyReqStatus  Displayed query for an Asynchronous Request in the current session.

  TyShutdown  Shut down the Tymeac Server.

  TyStats  On-Request statistics generation.

  TyStalled  Display/purge of stalled asynchronous requests.

  TyWlData  Display of a Queue's Wait Lists during execution.

   Samples / Demonstration

  Sample Classes

  com.tymeac.demo.TySvrComm  
Tymeac Server Communication Class for the Demonstration System. The user maintained Class containing the Methods for Client communication with the Tymeac Server.

  com.tymeac.demo.jframe.TyDemoT1 
Demonstration System single event Class.

  com.tymeac.demo.jframe.TyDemoT3 
Demonstration System multiple event Class.

  com.tymeac.demo.jframe.TyDemoInternal 
Demonstration System Internal Server testing.

  com.tymeac.demo.jini.client.TyDemoJiniT1 
Demonstration System Jini example Class.
