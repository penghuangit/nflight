Messages: Request Brokering Ty04

This section details the Request Brokering messages.

(P) Printed to System.out.println() (this is optional).
(L) Sent to the Tymeac Log.
(N) Sent to the Tymeac Notification Function.

010C (P) (L) (N)

'TymeacInternalCancel' Function is missing, CallBacks non-functioning.

Tymeac does callbacks in Synchronous Requests by using an in internal Tymeac Function (TymeacInternalCancel), internal Queue (TymeacInternalCancel) and internal Processing Application Class (com.tymeac.base.TymeacInternalCancel). All the internal Functions are loaded at Tymeac start up. Not being able to find the Function during request processing means that there is system damage.

The Server continues.

The calling request is given a general notification that the callback failed.
