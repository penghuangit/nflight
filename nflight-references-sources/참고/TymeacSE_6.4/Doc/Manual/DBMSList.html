DBMS Tymeac List Table

This table contains the List of Queue Names for each Function in the Tymeac system. All column names are UPPER CASE, must be in the order presented here, and must be spelled as presented here. Variable Character columns convert into Java Strings, therefore, lengths are the same as the DBMS Function Table and DBMS Queue Table. 

The purpose of this table is to associate Tymeac Queues with a Tymeac Function. Each Tymeac Function has from one to many Queues. This is where the List of Queues resides. 

There is no separate Tymeac Maintenance Class, the Function Maintenance Class, (TyFuncMaint), maintains this Table. For each Queue associated with this Function, the Class adds a row with a key of Function Name and a sequence number incremented from 1.

There is a pre-built definition in the /scripts/sql directory for this table to make your life a little easier.

Keys:  

Dual key-- FUNC_NAME, and SEQ_NBR.

Columns:

FUNC_NAME    Var Char.    The name of the Tymeac Function.

SEQ_NBR    Integer.   An increment starting from 1, used to keep the keys unique.        

QUEUE_NAME   Var Char.    A single Tymeac Queue associated with this Tymeac Function.
