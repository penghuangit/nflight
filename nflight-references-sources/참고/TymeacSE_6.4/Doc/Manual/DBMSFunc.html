DBMS Tymeac Function Table

This table contains all the Functions in the Tymeac system. All column names are UPPER CASE, must be in the order presented here, and must be spelled as presented here. Variable Character columns convert into Java Strings, therefore, lengths are customer determined. 

See the Tymeac Function Maintenance Class, (TyFuncMaint), for a description of values for each column.

There is a pre-built definition in the /scripts/sql directory for this table to make your life a little easier.

Keys:  

Single key -- FUNC_NAME

Columns:

FUNC_NAME    Var Char.    The name of a Tymeac Function.

AGENT_NAME    Var Char.    The name of an Output Agent Queue.   

NBR_QUEUES   Integer.     The number of Queues in this Function.                                                         (Internally generated.)
