DBMS Tymeac Statistics Table

This optional table contains the Tymeac System Statistics. Tymeac writes statistics at shut down and on request, TyStats. The format of the statistics is here.

All column names are UPPER CASE, must be in the order presented here, and must be spelled as presented here. Variable Character columns convert into Java Strings, therefore, lengths are customer determined. No Tymeac message is over 200 characters.

There is no separate Tymeac Maintenance Class, this table is user maintained.

The purpose of this table is to accept statistics generated by the Tymeac Server at shut down and on request.

The date/time stamp is from GregorianCalendar Methods.

There is a pre-built definition in the /scripts/sql directory for this table to make your life a little easier.

Keys:  

None

These name changed in release 5.1 with the addition of the underscore after the first initial. (e.g. YEAR became Y_EAR)

Columns:

Y_EAR    Integer.

M_ONTH   Integer.

D_AY    Integer.

H_OURS    Integer.

M_INUTES    Integer.

S_ECONDS    Integer.

M_ESSAGE    Var Char.    The Tymeac Server Statistic.
