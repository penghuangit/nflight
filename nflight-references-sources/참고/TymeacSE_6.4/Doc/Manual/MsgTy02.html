Messages: Shut Down Ty02

This section details the Shut Down messages.

(P) Printed to System.out.println(), (this is optional).
(L) Sent to the Tymeac Log.
(N) Sent to the Tymeac Notification Function.

001 I (P)

Tymeac Server Shutting down.

The Server finished stage one of the shut down sequence.  No new requests are accepted.  Currently executing request may finish.

009 I (P) (L)

Tymeac Server, Shut down complete for MilliTime= xx, duration=yyyy

The Server is shut down. 

xx is the Tymeac Server start up time in milliseconds as found on the start up message, Ty00001I.

yyyy is the Server up in hours:minutes:seconds

010 I (P) (L)

Tymeac Server, Shut down completed by force for MilliTime= xx, duration=yyyy

The Server is shut down.

 xx is the Tymeac Server start up time in milliseconds as found on the start up message, Ty00001I.

yyyy is the Server up in hours:minutes:seconds

HOWEVER, the shutdown was by force since Queue Threads were active.

020 I (P), (L)

Tymeac Server User Exit Shutdown Class <class name> Error=xxx

A User Exit Shutdown in either stage 1 or 2 Class <the name of the class> had an error defined as xxx.

Shut down  continues.

021 C (P), (L)

Tymeac Server User Exit Shutdown Class <class name> Invalid format

A User Exit Shutdown in either stage 1 or 2 Class <the name of the class> had a Class::URL format error. See the Class URL document

Shut down  continues.

022 C (P), (L)

Tymeac Server User Exit Shutdown Class <class name> Invalid URL Format

A User Exit Shutdown in either stage 1 or 2 Class <the name of the class> had a URL format error. See the Class URL document.

Shut down  continues.

025 I  (L)

Tymeac Server remote exception= (The String message for the Exception)

During an attempt to unregister the remote object from the Activation system, the method
        java.rmi.activation.Activatable.unregister();

threw a RemoteException.

Shut down  continues.

026 I  (L)

Tymeac Server activation exception= (The String message for the Exception)

During an attempt to unregister the remote object from the Activation system, the method
        java.rmi.activation.Activatable.unregister();

threw a ActivationException.

Shut down  continues.

027 I  (L)

Tymeac Server Unknown Object exception= (The String message for the Exception)

During an attempt to unregister the remote object from the Activation system, the method
        java.rmi.activation.Activatable.unregister();

threw a UnknownObjectException.

Shut down  continues.
