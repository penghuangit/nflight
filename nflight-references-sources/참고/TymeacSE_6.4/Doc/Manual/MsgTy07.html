Messages: Output Agent Thread Ty07

This section details the Output Agent Thread messages. These error messages reflect error codes found on the stalled asynchronous request display.

(P) Printed to System.out.println(), (this is optional).
(L) Sent to the Tymeac Log.
(N) Sent to the Tymeac Notification Function.

to access the Processing Application Class.

This exception was caught.

090 S (L) (P)

Thread [name] in Queue [name] Expunged

The Tymeac status of the thread was "disabled" (see how threads become disabled.)

An administrator used the ENABLE ALL button on the Queue Thread display to enable all disabled threads. This button sets the instance reference of the thread (in the thread management structure) to null. A new thread may or may not have taken the place of the thread writing this message.

When the thread writing this message resumes, it finds its instance reference either null or that another thread has taken its place, it writes this message and terminates quickly (exits in the run() method.)

901 S (L) (P) (N)

Thread [name] Failed with uncaught Exception=xxxx

This exception (xxxx) was caught by the uncaught exception handler.

The thread is logically disabled and the physical thread terminates.
