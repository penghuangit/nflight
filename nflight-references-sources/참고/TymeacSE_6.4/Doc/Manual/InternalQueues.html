Tymeac Internal Queues

The purpose of this Class is so the Server may execute internal Tymeac Functions. One such Function is the canceling of synchronous requests.

This class is required and is not maintainable by customers. Since Tymeac is Open Source, you have access to the source code. It is identical to the User Queues.

Tymeac start up loads this class after processing the User or DBMS Queues. Any errors are fatal.
