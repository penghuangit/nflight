Tymeac Sample Classes

This section explains the sample Classes we supply with the product. You may use Tymeac as a product without ever perusing the full source code. Certain modules are necessary for that end.

  The following source code modules are the security Classes.

com.tymeac.base.ServerSecurityModule
This Class contains the Server side LoginContext and login code for using a base security system.

com.tymeac.client.ClientSecurityModule
This Class contains the Client side LoginContext and login code for using a base security system.

com.tymeac.jini.base.JiniSecurityModule
This Class contains the Jini Server side LoginContext and login code for using a base security system.

  The following source code modules are the Run Time Classes.

com.tymeac.common.TyVariables
This Class contains the tymeac_name and URL_name used for RMI binding/naming and the RMI URL.

com.tymeac.demo.TySvrComm
This Class contains the methods used for communication with the Tymeac Server.

com.tymeac.serveruser.TyServerRMISecMgr
This Class loads the RMI Security Manager. See here for a description.

com.tymeac.serveruser.TymeacUserSingleton
Tymeac supplies a basic Singleton Class. See here for a description.

  The following source code modules are the User Classes.  These Classes are for a Stand-Alone system, that is -- one without a data base management system.

com.tymeac.serveruser.TymeacUserVariables
This Class maintains variables for Tymeac start up when no local file or DBMS is used.

com.tymeac.serveruser.TymeacUserQueues
This Class maintains Tymeac Queues when no local file or DBMS is used.

com.tymeac.serveruser.TymeacUserFunctions
This Class maintains Tymeac Functions when no local file or DBMS is used.

  The following source code is for customer use in validity checking the User Class Functions and Queues in the Utility directory:

FunctionChecker
This class has methods for checking the UserClass and DBMS Functions for duplicate names, duplicate hash codes and null entries.

QueueChecker
This class has methods for checking the UserClass and DBMS Queues for duplicate names and null entries.

com.tymeac.demo.TyDemoBaseRMI
This is run time example of how to execute the creation of an activatable Tymeac Server (the above Class, TyDemoActivationBase) and register that remote object with the Java Remote Object Registry.
