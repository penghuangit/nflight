New Run Time Logging

Logging is dynamically alterable with the New Run Time Log GUI/Non-GUI. During the session, if Tymeac finds a problem with logging, then Tymeac sets logging as unavailable. You may fix the problem and reset logging, stop all logging or set new logging parameters below.

Elements - each is mutually exclusive

DBMS Table  -- Enter the name of the New DBMS Table.

Dir and/or File Name  -- When using a local file instead of a DBMS Table, then enter the name of the file. A directory is optional

Alt  -- When using an alternate class instead of the above, enter that fully qualified class name.

Buttons

Reset -- Re-verifies the existing Logging on the Server. If all goes well, then Logging is restored.

Stop -- Stops all Logging on the Server.

Import -- Fetches the current Logging parameters from the Server.

Update -- Fill in a new Logging parameters and press to update on the Server. If you never had Logging at Server start-up, then you can add it now.

Message Block at bottom : -- (xx) is the return code for the non-gui class. This is an information message:

(-1)  Connection failure -- The RMI connection failed or the Tymeac Server returned an invalid message.

Since Tymeac is fully open-source now, you must go to the source code to find the return codes. All codes are public static final int values in Class com.tymeac.base.SetUpLog.java. As of Release 6.2 they are as follows:

public static final int LogSuccess = 0; // success
public static final int LogNoCurr = 20; // no current log
public static final int LogNotNec = 21; // log doesn't need resetting
public static final int LogWriteOldFail = 22; // write to original log failed
public static final int LogFailVeryOld = 23; // failed to verify original
public static final int LogCreateFail = 24; // New log failed to create
public static final int LogWriteNewFail = 25; // write to new log failed
public static final int LogFailVeryNew = 26; // failed to verify new
public static final int LogNotInUse = 27; // logging not in use
public static final int LogStopped = 28; // logging service stopped
